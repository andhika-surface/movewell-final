# Images Folder

This folder is for storing article and promo images.

## How to Upload Images

1. Go to GitHub repository
2. Navigate to `/data/images` folder
3. Click "Add file" → "Upload files"
4. Drag & drop your images
5. Commit changes

## Using Images in Articles/Promos

After uploading, reference images using:
```
/data/images/your-image-name.jpg
```

## Recommended Image Specs

- **Format:** JPG, PNG, or WebP
- **Size:** Max 1MB per image
- **Dimensions:** 
  - Article images: 1200x800px (3:2 ratio)
  - Promo images: 1200x600px (2:1 ratio)
- **Optimization:** Use tools like TinyPNG before uploading

## Image Naming Convention

Use descriptive names with hyphens:
- ✅ `massage-therapy-session.jpg`
- ✅ `opening-promo-banner.png`
- ❌ `IMG_1234.jpg`
- ❌ `image copy 2.jpg`
